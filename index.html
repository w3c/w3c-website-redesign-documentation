W3C website redesign project

Between 2020 and 2022, Studio 24 worked with the World Wide Web Consortium (W3C) on a redesign of w3.org to more effectively communicate what the organisation does with a modern, inclusive, usable website.

We restarted our work with W3C in May 2023, working on improvements and tackling community feedback, and the W3C website launched live in June 2023. You can continue give feedback to help us improve the W3C website.

We set this project site up to help work in the open, to share our progress, key documents and learnings.

You can view the full timeline or subscribe to updates via RSS.

Key deliverables

  • Live w3.org website
  • Beta w3.org website
  • Design system
  • HTML prototypes
  • Frontend website code repo
  • Craft CMS code repo (this repo is currently private)
