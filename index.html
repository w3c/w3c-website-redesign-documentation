W3C website redesign project

Between 2020 and 2022, Studio 24 worked with the World Wide Web Consortium (W3C) on a redesign of w3.org to more effectively communicate what the organisation does with a modern, inclusive, usable website.

We set this project site up to help work in the open, to share our progress, key documents and learnings. We strive to post regularly to this site, so sometimes documents will be draft and potentially work-in-progress.

You can view the full timeline or subscribe to updates via RSS.
